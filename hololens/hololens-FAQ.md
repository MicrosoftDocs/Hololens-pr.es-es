## <a name="my-holograms-dont-look-right-or-are-moving-around"></a>Mis hologramas no parecen correctos o se mueven

Si los hologramas no tienen el aspecto correcto (por ejemplo, están con vibración o vibración, o ve revisiones negras encima), pruebe una de estas correcciones:

- [Limpie el visor del dispositivo](hololens1-hardware.md#care-and-cleaning) y asegúrese de que no hay nada que bloquee los sensores.
- Asegúrese de que se encuentra en una sala bien iluminación que no tiene mucha luz directa.
- Pruebe a recorrer el entorno y a mirar su entorno para que HoloLens pueda examinarlos más por completo.
- Si ha colocado una gran cantidad de hologramas, intente quitar algunos.

Si sigue teniendo problemas, intente ejecutar la aplicación Calibración. Esta aplicación calibra tu HoloLens solo para que te ayudes a mantener los hologramas con el mejor aspecto posible. Para ello, vaya a Configuración  >  **Utilidades del**  >  **sistema**. En **Calibración,** seleccione **Abrir calibración.**

## <a name="i-see-a-message-that-says-finding-your-space"></a>Veo un mensaje que dice "Búsqueda del espacio".

Cuando HoloLens está aprendiendo o cargando un espacio, es posible que vea un breve mensaje que dice "Búsqueda del espacio". Si este mensaje se muestra durante más de unos segundos, verá otro mensaje en el menú Inicio que dice "Todavía buscando el espacio".

Estos mensajes significan que HoloLens tiene problemas para asignar el espacio. Cuando esto sucede, puede abrir aplicaciones, pero no puede colocar hologramas en su entorno.

Si ve estos mensajes con frecuencia, pruebe una o varias de las correcciones siguientes:

- Asegúrese de que se encuentra en una sala bien iluminación que no tiene mucha luz directa.
- Asegúrese de que el visor del dispositivo está limpio. [Obtenga información sobre cómo limpiar el visor](hololens1-hardware.md#care-and-cleaning).
- Asegúrese de que tiene una señal de Wi-Fi segura. Si escribe un nuevo entorno que no tiene Wi-Fi o una señal de Wi-Fi débil, HoloLens no podrá encontrar el espacio. Para comprobar Wi-Fi conexión, vaya **a** Configuración  >  **Red &amp;**  >  **Wi-Fi de Internet.**
- Intente moverse más lentamente.

## <a name="im-not-seeing-the-holograms-that-i-expect-to-see-in-my-space"></a>No veo los hologramas que esperaba ver en mi espacio

Si no ve los hologramas que ha colocado o si ve algunos que no espera, pruebe una o varias de las siguientes correcciones:

- Encender algunas luces. HoloLens funciona mejor en un espacio bien encendido.
- Quite los hologramas que no necesite; para ello, vaya a Configuración   >  **Hologramas** del  >  **sistema**  >  **Quitar hologramas cercanos.** O bien, si es necesario, **seleccione Quitar todos los hologramas.**

  > [!NOTE]
  > Si el diseño o la iluminación del espacio cambian significativamente, es posible que el dispositivo tenga problemas para identificar el espacio y mostrar los hologramas.

## <a name="i-cant-place-holograms-where-i-want-to"></a>No puedo colocar hologramas donde quiera

Estos son algunos aspectos que debe probar si tiene problemas para colocar hologramas:

- Se encuentra entre uno y tres metros desde donde está intentando colocar el holograma.
- No coloque hologramas en superficies negra o reflectante.
- Asegúrese de que se encuentra en una sala bien iluminación que no tiene mucha luz directa.
- Pasee por las salas para que HoloLens pueda volver a examinar su entorno. Para ver lo que ya se ha examinado, pulse en el aire para mostrar el gráfico de malla de asignación.

## <a name="holograms-disappear-or-are-encased-in-other-holograms-or-objects"></a>Los hologramas desaparecen o están incluidos en otros hologramas u objetos

Si se acerca demasiado a un holograma, desaparecerá temporalmente para restaurar el holograma, simplemente &mdash; aléjese de él. Además, si ha colocado varios hologramas juntos, algunos pueden desaparecer. Intente quitar algunos.

Los hologramas también se pueden bloquear o codificar mediante otros hologramas o por objetos como las paredes. Si esto sucede, pruebe una de las siguientes correcciones:

- Si el holograma está encerrado en otro holograma, mueva el holograma encasado a otra ubicación. Para ello, seleccione **Ajustar** y, a continuación, pulse y mantenga presionado para colocarlo.
- Si el holograma está encerrado en una pared, seleccione **Ajustar** y, a continuación, avance hacia la pared hasta que aparezca el holograma. Pulse y mantenga presionado y, a continuación, extraiga el holograma hacia delante y fuera de la pared.
- Si no puedes mover el holograma mediante gestos, usa tu voz para quitarlo. Mire el holograma y, a continuación, diga "Quitar". A continuación, vuelva a abrir el holograma y colóctelo en una nueva ubicación.

## <a name="i-can-see-holograms-that-are-on-the-other-side-of-a-wall"></a>Puedo ver hologramas que están al otro lado de una pared

Si está muy cerca de una pared o si HoloLens aún no ha examinado la pared, puede ver los hologramas que se encuentran en la siguiente sala. Para examinar la pared, de pie entre uno y tres metros de la pared y mire hacia ella.

Un objeto negro o reflectante (por ejemplo, un cómodo negro o un refrigerador de cristal) cerca de la pared puede causar problemas cuando HoloLens intenta examinar la pared. Si hay un objeto de este tipo, escalo el otro lado de la pared.

## <a name="when-i-place-a-hologram-on-a-wall-the-hologram-seems-to-float"></a>Cuando se coloca un holograma en una pared, el holograma parece flotar.

Normalmente, un holograma que se coloca en una pared parece estar a una pulgada o así de la pared. Si parece que está más lejos, pruebe una o varias de las correcciones siguientes:

- Cuando coloque un holograma en una pared, se coloca entre uno y tres metros de la pared y se enfrenta directamente a la pared.
- Pulse el aire en la pared para mostrar el gráfico de malla de asignación. Asegúrese de que la malla se alinea con la pared. Si no es así, quite el holograma, vuelva a examinar la pared e inténtelo de nuevo.
- Si el problema persiste, ejecute la aplicación Calibración. Lo encontrará en Settings **System** Utilities  >  **(Utilidades del**  >  **sistema de configuración).**

## <a name="im-getting-a-low-disk-space-error"></a>Obtención de un error de poco espacio en disco

Para liberar espacio de almacenamiento, haga uno o varios de los siguientes pasos:

- Quite algunos de los hologramas que ha colocado o quite algunos datos guardados de las aplicaciones. [Cómo buscar mis datos?](holographic-data.md)
- Elimine algunas imágenes y vídeos en la aplicación Fotos.
- Desinstale algunas aplicaciones de HoloLens. En la **lista Todas las** aplicaciones, mantenga presionada la aplicación que desea desinstalar y, a continuación, seleccione **Desinstalar**. (Al desinstalar la aplicación también se eliminan los datos que almacena la aplicación en el dispositivo).

## <a name="hololens-doesnt-respond-to-my-gestures"></a>HoloLens no responde a mis gestos

Para asegurarse de que HoloLens puede ver los gestos, mantenga la mano en el marco de gestos. El marco de gestos extiende un par de pies a cada lado. HoloLens también puede ver mejor la mano cuando se mantiene unas 18 pulgadas delante del cuerpo (aunque no es preciso ser preciso al respecto). Cuando HoloLens puede ver la mano, el cursor cambia de un punto a un anillo. Obtenga más información [sobre el uso de gestos HoloLens 2](hololens2-basic-usage.md) o el uso de [gestos en HoloLens (1.ª generación).](hololens1-basic-usage.md)

## <a name="hololens-doesnt-respond-to-my-voice"></a>HoloLens no responde a mi voz

HoloLens (1.ª generación) y HoloLens 2 reconocimiento de voz integrado y también admiten Cortana (reconocimiento de voz en línea).

### <a name="built-in-voice-commands-do-not-work"></a>Los comandos de voz integrados no funcionan

En HoloLens (1ª generación), el reconocimiento de voz integrado no es configurable. Siempre está activado. En HoloLens 2, puede elegir si quiere activar el reconocimiento de voz y Cortana durante la configuración del dispositivo.

Si el HoloLens 2 no responde a su voz, asegúrese de que el reconocimiento de voz está activado. Vaya a **Configuración de inicio**  >  **Privacidad**  >    >  **Voz** y active Reconocimiento **de voz.**

### <a name="cortana-or-dictation-doesnt-work"></a>Cortana o dictado no funciona

Si Cortana o Dictado no responden a su voz, asegúrese de que el reconocimiento de voz en línea está activado. Vaya a **Configuración de inicio**  >  **Privacidad**  >    >  **Speech** y compruebe la configuración **del reconocimiento de voz en** línea. 

Si Cortana sigue sin responder, realice una de las siguientes acciones para comprobar que el propio Cortana está activado:

- En **Todas las aplicaciones,** seleccione **Cortana** > **Menú Configuración** del  >  **cuaderno** para realizar  >   cambios.
- En HoloLens 2, seleccione el botón **Configuración de** voz o diga "Configuración de voz".

Para más información sobre lo que puede decir, consulte [Uso de la voz con HoloLens.](hololens-cortana.md)

## <a name="hololens-settings-lists-devices-as-available-but-the-devices-dont-work"></a>Configuración de HoloLens muestra los dispositivos como disponibles, pero los dispositivos no funcionan

HoloLens (1.ª generación) no admite perfiles de audio Bluetooth. Los dispositivos de audio Bluetooth, como altavoces y cascos, pueden aparecer como disponibles en la configuración de HoloLens, pero no se admiten.

HoloLens 2 admite el perfil de audio Bluetooth A2DP para la reproducción estéreo. El perfil Bluetooth Hands Free que permite la captura de micrófono desde un periférico Bluetooth no se admite en HoloLens 2.

Si tiene problemas para usar un dispositivo Bluetooth, asegúrese de que es un dispositivo compatible. Entre los dispositivos admitidos se incluyen los siguientes:

- Teclados Bluetooth QWERTY en inglés (puede usarlos en cualquier lugar en el que use el teclado holográfico).
- Mouse Bluetooth.
- El [botón de clic de HoloLens](hololens1-clicker.md).

Puede emparejar otros dispositivos BLUETOOTH HID y HOLOLens junto con hololens. Sin embargo, es posible que tenga que instalar las aplicaciones complementarias correspondientes Microsoft Store usar realmente los dispositivos.

## <a name="im-having-problems-using-the-hololens-clicker"></a>Tengo problemas al usar el clic de HoloLens

Use el [clicker](hololens1-clicker.md) para seleccionar, desplazar, mover y cambiar el tamaño de los hologramas. Las aplicaciones individuales pueden admitir gestos de clic adicionales.

Si tiene problemas para usar el clicker, asegúrese de que se cobra y se empareja con HoloLens. Si la batería está baja, la luz del indicador parpadea en ámbar. Para comprobar que el botón de clic está emparejado, vaya a **Dispositivos** de configuración y vea  >   si aparece allí. Para obtener más información, vea [Emparejar el botón de clic.](hololens1-clicker.md)

Si el clic se cobra y empareja y sigue teniendo problemas, restablezcalo manteniendo presionado el botón principal y el botón de emparejamiento durante 15 segundos. A continuación, vuelva a emparejar el clicker con holoLens.

Si restablecer el clicker no ayuda, consulta Reiniciar o recuperar el clic [de HoloLens.](hololens1-clicker.md#restart-or-recover-the-clicker)

## <a name="i-cant-connect-to-wi-fi"></a>No puedo conectarme a Wi-Fi

Estos son algunos aspectos que se pueden probar si no se puede conectar HoloLens a una Wi-Fi virtual:

- Asegúrese de que Wi-Fi está activado. Para comprobarlo, use el gesto Iniciar y, a continuación, seleccione **Configuración**  >  **&amp; Red**  >  **Wi-Fi de Internet.** Si Wi-Fi está encendido, intente desactivarlo y, a continuación, vuelva a activarlo.
- Acercarse al enrutador o al punto de acceso.
- Reinicie el enrutador Wi-Fi y, a [continuación, reinicie HoloLens.](hololens-recovery.md) Try connecting again (Intente conectarse de nuevo).
- Si ninguna de estas cosas funciona, asegúrese de que el enrutador usa el firmware más reciente. Puede encontrar esta información en el sitio web del fabricante.

## <a name="i-cant-sign-in-to-a-hololens-device-because-it-was-previously-set-up-for-someone-else"></a>No puedo iniciar sesión en un dispositivo HoloLens porque se ha configurado previamente para otra persona.

Si el dispositivo se ha configurado previamente para otra persona, ya sea para un cliente o para un empleado anterior, y no tiene su contraseña para desbloquear el dispositivo, puede realizar una de las siguientes acciones:

- Para un dispositivo inscrito en la administración de dispositivos móviles (MDM) de Intune, puede usar Intune para borrar [el](https://docs.microsoft.com/intune/remote-actions/devices-wipe) dispositivo de forma remota. A continuación, el dispositivo se vuelve a parpadear.  
   > [!IMPORTANT]  
   > Al borrar el dispositivo, asegúrese de dejar desactivada la opción Conservar el estado **de inscripción y la cuenta** de usuario.
- En el caso de un dispositivo que no es MDM, puede poner el dispositivo en modo de flashing y usar Advanced [Recovery Companion  ](hololens-recovery.md#clean-reflash-the-device) para recuperar el dispositivo.

## <a name="questions-about-managing-hololens-devices"></a>Preguntas sobre la administración de dispositivos HoloLens

### <a name="can-i-use-system-center-configuration-manager-sccm-to-manage-hololens-devices"></a>¿Puedo usar System Center Administrador de configuración (SCCM) para administrar dispositivos HoloLens?

No. Debe usar un sistema MDM para administrar dispositivos HoloLens.

### <a name="can-i-use-active-directory-domain-services-ad-ds-to-manage-hololens-user-accounts"></a>¿Puedo usar Active Directory Domain Services (AD DS) para administrar cuentas de usuario de HoloLens?

No. Tiene que usar Azure Active Directory (Azure AD) para administrar cuentas de usuario para dispositivos HoloLens.

### <a name="is-hololens-capable-of-automated-data-capture-systems-adcs-auto-enrollment"></a>¿HoloLens es capaz de la inscripción automática de sistemas de captura de datos automatizados (ADCS)?

No.

### <a name="can-hololens-participate-in-integrated-windows-authentication"></a>¿HoloLens puede participar en Autenticación integrada de Windows?

No.

### <a name="does-hololens-support-branding"></a>¿HoloLens admite la personalción de marca?

No. Sin embargo, puede evitar este problema mediante uno de los enfoques siguientes:

- Cree una aplicación personalizada y, a continuación, [habilite el modo quiosco.](hololens-kiosk.md) La aplicación personalizada puede tener personalización de marca y puede iniciar otras aplicaciones (como Remote Assist).  
- Cambie todas las imágenes de perfil de usuario de Azure AD por el logotipo de su empresa. Sin embargo, esto puede no ser deseable para todos los escenarios.

### <a name="what-logging-capabilities-do-hololens-1st-gen-and-hololens-2-offer"></a>¿Qué funcionalidades de registro ofrecen HoloLens (1ª generación) y HoloLens 2 generación?

El registro se limita a los seguimientos que se pueden capturar en escenarios de desarrollo o solución de problemas, o telemetría que los dispositivos envían a los servidores de Microsoft.
